### 2. 🛡️ WAF Defense Validation — ModSecurity + Apache
**Deployed ModSecurity in blocking mode** with custom rules targeting SQL Injection and XSS. Attacks confirmed successful pre-WAF, then fully neutralised post-deployment across DVWA and PrestaShop. All 4 attacks blocked with HTTP 403.

`ModSecurity` · `Apache` · `OWASP CRS` · `SQLi` · `XSS` · `DVWA` · `PrestaShop` · `curl`

🔗 **[View Project →](https://github.com/ChuksOko/-WAF-Defense-Validation-Lab---ModSecurity-Apache)**

### 3. 🔍 SSH Brute-Force Detection — ELK Stack SIEM
**Deployed full ELK Stack SIEM**, ingested syslog data, simulated an SSH brute-force attack using Hydra, detected the attack in real time in Kibana Discover, and built a custom SSH Brute Force Detection Dashboard. Includes AI adversarial risk assessment of Hydra.

`Elasticsearch` · `Logstash` · `Kibana` · `Hydra` · `KQL` · `Syslog` · `Ubuntu`

🔗 **[View Project →](https://github.com/ChuksOko/ELK-SSH-Bruteforce-Detection)**

### 4. ☁️ Cloud Privilege Escalation Simulation — AWS IAM

`AWS IAM` · `AWS CLI` · `PowerShell` · `Privilege Escalation` · `Least Privilege` · `MITRE ATT&CK T1098`
